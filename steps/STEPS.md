#Steps to config a development environment to work with React.js

1.  Setup tools for project standards

  - ESlint
  - Prettier

  ## Installing

  `npx eslint --init`
  `npm install --save-dev --save-exact prettier`

  - Create Prettierrc file (.prettierrc) with config:
  ```{
  "trailingComma": "es5",
  "tabWidth": 2,
  "semi": true,
  "singleQuote": true
}```


2.  Setup Loaders

- Babel
- Style Loaders
- File Loaders

## Installing

`npm install`
`npm install --save-dev --save-exact prettier`

3.  Setup Plugins

- HTML Plugin

## Installing

`npm install`

4.  Setup Webpack

- Configure Webpack.config file
