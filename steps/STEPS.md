# React Development ENV

## 1. Tools for project standards

  - ### ESlint
  - ### Prettier

  ## Installation

  ### ESLint
  
  `npx eslint --init`

  Obs: Accept all config to set Airbnb Standards

  ### Prettier

  `npm install --save-dev --save-exact prettier`

  ### Create Prettier file (.prettierrc) with the following config:
  
```
{
  "trailingComma": "es5",
  "tabWidth": 2,
  "semi": true,
  "singleQuote": true  
}
```
